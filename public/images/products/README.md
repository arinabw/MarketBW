# Изображения продуктов

В этой папке должны находиться изображения изделий.

## Структура изображений для каждого продукта:

```
products/
├── bracelet-1-1.jpg    # Основное фото браслета 1
├── bracelet-1-2.jpg    # Дополнительное фото браслета 1
├── bracelet-1-3.jpg    # Дополнительное фото браслета 1
├── necklace-1-1.jpg    # Основное фото колье 1
├── necklace-1-2.jpg    # Дополнительное фото колье 1
├── earrings-1-1.jpg    # Основное фото сережек 1
├── earrings-1-2.jpg    # Дополнительное фото сережек 1
├── brooch-1-1.jpg      # Основное фото броши 1
├── brooch-1-2.jpg      # Дополнительное фото броши 1
├── bracelet-2-1.jpg    # Основное фото браслета 2
├── bracelet-2-2.jpg    # Дополнительное фото браслета 2
└── necklace-2-1.jpg    # Основное фото колье 2
└── necklace-2-2.jpg    # Дополнительное фото колье 2
└── necklace-2-3.jpg    # Дополнительное фото колье 2
```

## Требования к изображениям:

- **Формат**: JPG или PNG
- **Размер**: 800x800px для основных фото, 400x400px для дополнительных
- **Качество**: Высокое, четкие фотографии на светлом фоне
- **Стиль**: Романтичный, нежный, с хорошим освещением
- **Фон**: Светлый, однородный (белый, светло-розовый, лавандовый)

## Текущие заглушки:

Сейчас используются файлы-заглушки. Замените их на реальные фотографии ваших изделий из бисера.

# План сайта для изделий из бисера

## Обзор проекта
Сайт для продажи и демонстрации украшений из бисера (браслеты, колье, серьги, броши) с романтичным и нежным дизайном в пастельных тонах.

---

## Структура сайта

### Основные страницы

#### 1. Главная страница (Home)
- **Hero секция**: Приветствие, красивое фото лучших изделий, призыв к действию
- **Категории**: Быстрый доступ к типам украшений (браслеты, колье, серьги, броши)
- **Популярные изделия**: Карусель с самыми популярными работами
- **Отзывы**: Несколько отзывов от клиентов
- **Контакты**: Быстрые способы связи

#### 2. Каталог (Catalog)
- **Фильтры**: По типу изделия, цене, цвету, стилю
- **Сетка изделий**: Карточки с фото, названием, ценой
- **Сортировка**: По цене, новизне, популярности
- **Пагинация**: Для удобной навигации

#### 3. Страница изделия (Product Detail)
- **Галерея фото**: Основное фото + дополнительные виды
- **Описание**: Подробное описание изделия
- **Характеристики**: Размеры, материалы, техника плетения
- **Цена**: Чёткое указание стоимости
- **Кнопка заказа**: Форма заказа или связь через мессенджеры
- **Похожие изделия**: Рекомендации

#### 4. Контакты (Contact)
- **Форма обратной связи**
- **Контактная информация**: Телефон, email, адрес
- **Социальные сети**: Ссылки на Instagram, VK, Telegram
- **Карта**: Если есть физическая точка

#### 5. FAQ (Часто задаваемые вопросы)
- **Вопросы о заказе**: Как заказать, сроки, доставка
- **Вопросы об уходе**: Как ухаживать за изделиями
- **Вопросы о материалах**: Какие бусины используются

## Дизайн-концепция

### Цветовая палитра (Романтичный и нежный стиль)

**Основные цвета:**
- `#FFF5F5` - Очень светлый розовый (фон)
- `#FFE4E1` - Мисти роз (акценты)
- `#E6E6FA` - Лавандовый (вторичные элементы)
- `#F8F8FF` - Призрачный белый (карточки)

**Текстовые цвета:**
- `#4A4A4A` - Тёмно-серый (основной текст)
- `#6B6B6B` - Средне-серый (вторичный текст)
- `#D4A5A5` - Пастельный розовый (заголовки)

**Акцентные цвета:**
- `#C9A9A6` - Пыльная роза (кнопки, CTA)
- `#B8A9C9` - Пастельный фиолетовый (подсветка)

### Типографика

**Заголовки:**
- Шрифт: Playfair Display (с засечками, элегантный)
- Размеры: H1 - 48px, H2 - 36px, H3 - 28px

**Основной текст:**
- Шрифт: Lato (без засечек, читаемый)
- Размер: 16px для основного текста, 14px для вторичного

### Визуальные элементы

**Формы:**
- Скруглённые углы (border-radius: 12px для карточек)
- Мягкие тени (box-shadow: 0 4px 20px rgba(0,0,0,0.08))
- Плавные переходы (transition: all 0.3s ease)

**Иконки:**
- Использование Feather Icons или Lucide Icons
- Тонкие, изящные линии

**Изображения:**
- Высококачественные фотографии изделий
- Мягкое освещение
- Нейтральный фон для фото изделий

### Анимации

- Плавное появление элементов при скролле (fade-in)
- Hover эффекты на карточках изделий (лёгкое увеличение)
- Плавные переходы между страницами
- Микроанимации на кнопках

## Технологический стек

### Frontend Framework
**Next.js 14** (App Router)
- Современный React-фреймворк
- Server-Side Rendering для SEO
- Отличная производительность
- Встроенная оптимизация изображений

### UI Components
**shadcn/ui**
- Современная библиотека компонентов
- Полностью кастомизируемая
- Отличная документация
- Совместима с Tailwind CSS

### Стилизация
**Tailwind CSS**
- Utility-first CSS фреймворк
- Быстрая разработка
- Отличная производительность
- Легкая кастомизация

### Анимации
**Framer Motion**
- Декларативные анимации
- Плавные переходы
- Отличная производительность

### Иконки
**Lucide React**
- Современные иконки
- Лёгкие и кастомизируемые
- Отличная документация

### Формы
**React Hook Form + Zod**
- Валидация форм
- Отличная производительность
- TypeScript поддержка

### Оптимизация изображений
**Next.js Image Optimization**
- Автоматическая оптимизация
- Lazy loading
- WebP формат
